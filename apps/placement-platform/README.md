# Placement Readiness Platform

A React-based platform for placement preparation with practice problems, mock interviews, and progress tracking.

## Features

- **Landing Page**: Hero section with call-to-action and features grid
- **Dashboard**: Full-featured dashboard with sidebar navigation
- **Practice**: Coding problems and challenges
- **Assessments**: Mock tests and assessments
- **Resources**: Learning materials and study resources
- **Profile**: User profile and settings

## Tech Stack

- React 18
- React Router DOM
- Tailwind CSS
- Lucide React (icons)
- Vite (build tool)

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```
